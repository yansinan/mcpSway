# mcpSway

Sway 桌面环境相关的配置、脚本、技能仓库。

## 目录结构

```
mcpSway/
├── AGENTS.md      # AI Agent 工作指南
├── PRINCIPLES.md  # 仓库基本原则
├── skills/        # Hermes 技能定义（SKILL.md）
├── scripts/       # 实用脚本（sway-session 等）
├── configs/       # 参考配置（waybar、sway、wofi 等）
├── docs/          # 文档与笔记
└── README.md
```

## 目标

将 Sway 桌面的各种设置、脚本、技能集中管理，形成可维护、可复用的知识库。

### 计划收录的内容

- sway 配置文件参考（layout、keybindings、rules）
- waybar 配置（双栏 macOS 风格）
- sway-session 管理脚本（save/restore/daemon/idle）
- wofi / rofi 启动器配置
- mako 通知配置
- systemd 用户服务（uxplay、waybar 等）
- PipeWire / WirePlumber 音频配置
- 输入法（fcitx5）配置
- 蓝牙设备管理
- DPMS / s2idle 电源管理
