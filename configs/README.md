# configs/

Sway 及各组件的参考配置文件。

目录建议：
- `sway/`        — sway 配置（config, bar, rules 等）
- `waybar/`      — waybar 配置（config-top, config-bottom, style.css）
- `wofi/`        — 启动器配置
- `mako/`        — 通知配置
- `fcitx5/`      — 输入法配置
- `systemd/`     — user service 单元文件
- `environment/` — 环境变量配置（~/.config/environment.d/）
